# TechAI Assistant

An AI-powered computer diagnostics and solutions platform built with Databutton.

## Features

- 🔍 System Performance Analysis
- 🛡️ Security Analysis
- 💻 Hardware Diagnostics
- ⚡ Quick Resolution

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Shadcn UI Components

### Backend
- FastAPI
- Python
- DeepSeek AI
- psutil
